# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **hi@santifer.io**

Include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact
- Suggested fix (if you have one)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix or mitigation**: As soon as reasonably possible

## Scope

This policy applies to the `career-ops` repository and its dependencies.

## Supported Versions

Only the latest version on the `main` branch is actively maintained.

| Version | Supported |
|---------|-----------|
| main    | Yes       |
| Others  | No        |
